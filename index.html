AeroPlace-Flow: Language-Conditioned Object Placement for Aerial Manipulators via Visual Foresight and Object Flow


Abstract

Aerial manipulators enable robotic interaction in cluttered and elevated environments, but precise object placement remains challenging due to perception uncertainty and complex aerial dynamics. We present AeroPlace-Flow, a framework that performs language-conditioned object placement for aerial manipulators by leveraging generative visual foresight and metric object flow inference. Given a language instruction and scene observation, AeroPlace-Flow first generates a visual foresight of the desired placement configuration using off-the-shelf image editing models. We then recover the metric displacement of the manipulated object through object flow estimation, enabling the aerial manipulator to execute a smooth placement trajectory. We evaluate our method across diverse language-conditioned placement tasks and demonstrate reliable transfer from simulation to real-world aerial manipulation.
